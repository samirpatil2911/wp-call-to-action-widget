== Description ==

A Call to Action WordPress plugin that lets you place your CTAs within posts, pages and widget areas

**Requirements:** WordPress 4.0+, PHP5.3 and higher

You canâ€™t afford readers missing your CTAs. Place them at all the prominent areas within your site with WP Call to Action. You can add it to posts, pages and widget areas. You can add image, tile, subtitle, button. Choose between button type.

This plugin is free to use and is not actively supported by the author, but will be monitored for serious bugs that may need corrections.

== Installation ==

1. Upload the 'wp-call-to-action-widget' folder and all its contents to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance -> Widgets. Drag WP Call To Action Widget and drop in any widget area. Choose the settings and save. You are done!
